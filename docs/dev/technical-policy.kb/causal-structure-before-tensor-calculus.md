For GR/cosmology musings, reach for causal/conformal structure
(Penrose diagrams, light-cone reasoning) before reaching for tensor
calculus; the latter is forced only when dynamics or quantitative
questions arrive. Grounds: `C.CAUSAL-FIRST` (ratified).
