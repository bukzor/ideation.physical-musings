# Design tower

Layered normative documentation per Skill(llm-design-kb): each layer
justifies the one below via `why:` links upward.

`025-use-cases.kb/` is a local layer insertion between goals and
requirements: the musings the system must serve. They refine G.DIS and
inform requirements.

Layers 040-design and below are intentionally absent: durable design
derives strictly from settled claims (TP.DESIGN-FROM-SETTLED), and not
enough claims have settled yet.
