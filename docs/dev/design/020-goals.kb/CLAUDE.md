# Goals

Top-level `G.*` goals: stable, user-stated, aspirational-but-achievable.
Direct inputs, not derived from claims -- the user edits these at will;
they don't require ratification the way claims/requirements do.

One file per goal. `why:` points to `010-mission.kb/mission.md`.
Desiderata/success conditions belong in `../030-requirements.kb/`, not
here.
