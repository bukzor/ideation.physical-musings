Build fully-internalized intuition, not just a checked record. The
system should both *stand in for* memory that hasn't internalized yet
and *help build toward* internalization -- see
[user-memory-decay-profile] for the empirical basis (recall decays from
full detail, to conclusions-only, to durable intuition, over different
timescales). Not a feature awaiting a dedicated mechanism: a design rule
already expressed through other choices ([python-schematic-twin],
[dual-representation-refinement], [representation-play]) -- clarified by
the user 2026-07-03.
