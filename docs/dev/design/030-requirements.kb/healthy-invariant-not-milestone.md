G.SYS is *healthy*, never *achieved* -- an invariant maintained under
growth, not a milestone. At every point: each settled claim carries
inspectable grounds (`R.SETTLE-AUDIT`); new sessions neither silently
contradict nor re-litigate it; reopening is always permitted but must
present something new. Metric: regressions + re-litigations per
session, held near zero as the corpus grows -- green CI, not a shipped
release. The rate itself is a live G.CONTIMP metric once instrumented
(`Q.BASELINE` covers establishing a before/after baseline from prior
transcripts).

User agreed to the substance; this wording is proposed, not yet
explicitly ratified.
