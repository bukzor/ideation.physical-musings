Confident subtle errors -- sign conventions, index gymnastics,
coordinate artifacts mistaken for physical effects, misapplied limits --
must be caught mechanically, not merely discussed away. Negation of
failure mode F.SUBTLE. Mechanical checking substrate is JIT
(`Q.FORMAL-SUBSTRATE`, `Q.CAS-CAN`, `Q.CAS-SHOULD`).
