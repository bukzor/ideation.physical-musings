Claims need a machine-checkable representation. Interior semantics can
be as nebulous or as rigorous as we see fit -- the checker verifies the
*discourse structure* (does this arrow follow from these premises), not
the physics itself. See `TP.LOCAL-CHECK` for why the check must be
local rather than global.
