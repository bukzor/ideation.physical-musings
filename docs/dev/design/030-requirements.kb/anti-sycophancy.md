The LLM must not agree with a plausible-but-wrong idea merely because
it's plausible or because the user pushes for agreement. Negation of
failure mode F.SYCOPHANCY. `TP.ADV-DISCUSS-ONLY` is one candidate
countermeasure, committed for this discussion's scope only -- not yet
decided whether it belongs in the eventual G.SYS design.
