# Use cases

`M.*` musings: the user's own ideas that G.DIS must be able to discuss
and G.SYS must be able to hold rigorously. A local layer between goals
and requirements (design tower insertion, not a stock design-kb layer).

Elaborate a musing into `$slug.kb/` when it has enough named
sub-variants to need one file each (see `cosmo-bc.kb/`). Physics
substance itself (claims, deductions, sources) does not live here --
that's a future per-musing discourse graph at repo root; these files
are scope-setting, not the discussion.
