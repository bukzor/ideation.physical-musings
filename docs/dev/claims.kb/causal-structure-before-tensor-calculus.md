[horizon-symmetries], [de-sitter-horizon-observer-dependence],
[alice-bob-schwarzschild-reconciliation]) live in causal/conformal
structure -- horizons, null surfaces, conformal infinity -- which is
rigorously discussable via Penrose diagrams and light-cone reasoning
without tensor calculus. Tensor machinery is only forced once
dynamics/quantitative questions arrive. Grounds
[technical-policy.kb/causal-structure-before-tensor-calculus].
