Model diversity is a non-issue: Claude for the foreseeable future, with
occasional ollama/aistudio excursions possible; neither party expects
ensembling to matter. Assistant's supporting argument: a second LLM is a
*correlated* witness, not an independent one -- shared training corpus
means shared dogma, exactly the blind spot [merit-over-authority]
targets -- so the decorrelated second opinion must come from mechanical
checks (CAS/ITP), not another model. Resolves
[should-we-cross-check-other-llms].
