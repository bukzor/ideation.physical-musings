The user's own recall decays on a schedule: full session detail for
about 2 weeks, "big big" conclusions (without the how/why) for about 6
months, and fully internalized intuition indefinitely. As an ideal,
G.SYS both stands in for this memory before it internalizes and helps
build toward internalization (`G.INTUIT`). Grounds `R.ANTI-AMNESIA`'s
refinement and `G.INTUIT`.
