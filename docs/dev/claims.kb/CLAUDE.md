# Claims (meta discourse)

Truth-apt assertions about *the system itself* -- G.SYS's shape, scope
decisions for this discussion, and facts about the user/process.
Physics claims belong in the per-musing discourses at repo root.

`ratified: <date>` marks a claim as settled per R.SETTLE-AUDIT; absence
means asserted (stated or proposed) but not yet through the ritual.
