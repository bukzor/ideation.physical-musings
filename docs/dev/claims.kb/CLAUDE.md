# Claims (meta discourse)

Truth-apt assertions about *the system itself* -- G.SYS's shape, scope
decisions for this discussion, and facts about the user/process. This is
one of two distinct discourses this project will host: this one is
*about the system*; a second kind (opened per musing, at repo root) is
about physics, with its own `sources.kb/`, `deductions.kb/`, etc.

`ratified: <date>` marks a claim as settled per R.SETTLE-AUDIT; absence
means asserted (stated or proposed) but not yet through the ritual.
