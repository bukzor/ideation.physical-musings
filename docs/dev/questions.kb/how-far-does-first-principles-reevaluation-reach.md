How far does first-principles re-evaluation ([merit-over-authority])
reach -- re-derive mainstream results before using them as premises, or
reserve skepticism for interpretive/dogmatic territory? Resolved by
dissolving the dichotomy: it's a posture, not a workload. Established
results are held loosely -- easy to reopen, defended on merits -- but
re-derivation is not the default activity.
