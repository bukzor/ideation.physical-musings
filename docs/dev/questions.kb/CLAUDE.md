# Questions (meta discourse)

Open inquiries about the system itself. `resolved:` points to the claim
that answers it; `candidate-resolutions:` for claims under
consideration. An unresolved question with a stated working default
(noted in its body) is still open -- the default just means the answer
isn't blocking progress meanwhile. Answered propositions belong in
`../claims.kb/`, not here -- move a question there only by resolving it,
never by converting it in place.
